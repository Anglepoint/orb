orb

Pivot grid widget javascript library
Rendering powered by React

Features

Main
  • >   Fields drag'n'drop
  • >   Multi data fields support
  • >   Grand totals & Sub totals
  • >   Sub totals expand/collapse
  • >   Sorting
 Configuration
  • -   Data headers location (row/column)
  • -   Sub totals (by field/row/column):
    • ::   Visibility
    • ::   Collapsible
    • ::   Initial collapsed state
  • -   Grand totals visibility by row/column
  • -   Data cells:
    • ::   Aggregate function
    • ::   Format function
